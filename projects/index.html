百变罪犯 游戏
《百变罪犯》是一款快节奏寻物、像素风格的休闲游戏。游戏规则简单但超级费眼睛，一边要看着线索尽快在几秒内抓到罪犯从而得高分评"S"，一边要小心不能选到无辜的路人，因为失败就要从头开始。
  • Unity
  • Node.js
  • Redis
暴打太空鼠 游戏
3D太空版打地鼠，旋转的星球不断冒出这些烦人的太空鼠，快去重重敲击他们吧！在倒计时结束之前打越多的太空鼠可以取得越高的分数，小心不要打到炸弹！真人排行榜看看你是不是一个没有感情的打地鼠机器～
  • Unity
  • Node.js
  • Redis
excel2flatbuffers-rs 工具
一个用于游戏开发配置表数据的小工具。将Excel表结构转换成 flatbuffers 结构，生成目标语言代码，并将 Excel 数据打包成 flatbuffers 二进制数据文件。这样在游戏中就可以使用 flatbuffers 解析配置数据。
  • Rust
  • Unity
  • Flatbuffers
